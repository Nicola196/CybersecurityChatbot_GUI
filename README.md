## 🖥️ Graphical User Interface (GUI)

The Cybersecurity Chatbot provides an intuitive graphical user interface (GUI) developed using Java Swing. The interface enables users to interact with the chatbot without requiring command-line knowledge.

### GUI Features

- Interactive chat window
- User-friendly navigation
- Text input field for questions
- Scrollable conversation history
- Responsive chatbot replies
- Easy access to cybersecurity information

### Main Screens

- Welcome Screen
- Main Chat Interface
- Help/Information Section
- Exit Option

---

## 💬 Example Conversation

**User**
```
What is phishing?
```

**Bot**
```
Phishing is a cyberattack where attackers impersonate legitimate organizations to steal sensitive information such as usernames, passwords, and banking details.
```

---

**User**
```
How can I create a strong password?
```

**Bot**
```
Use at least 12 characters including uppercase letters, lowercase letters, numbers, and special symbols. Avoid using personal information.
```
